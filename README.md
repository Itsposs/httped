### 最初的想法
本项目是我在杭州正吾资产投资顾问有限公司实习期间利用晚上和周末的时间完成的，实习期间我负责交易系统的交易记录读取算法设计与实现，我希望能把其中的通信部分的成果通过Web的方式展示出来，但因为涉及保密协议，不得不放弃这一想法。
